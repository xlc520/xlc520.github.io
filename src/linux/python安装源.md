# python 安装源

在`C:\Users\Administrator\pip`下，新建`pip.ini`，输入以下内容：

```python
[global]
timeout = 6000
index-url = https://mirrors.aliyun.com/pypi/simple/
[install]
use-mirrors = true
mirrors = https://mirrors.aliyun.com/pypi/simple/
trusted-host = mirrors.aliyun.com
```
